# Change Log

All notable changes to the "Go Swaggo Highlight" extension will be documented in this file.

## [Unreleased]

- Added godoc-gated semantic highlighting for Swaggo annotations
- Added document and workspace symbol resolution for dynamic type and variable coloring
- Added cache layers for document and workspace symbol lookups
- Expanded publishing metadata and project documentation

## [0.0.2]

- Improved semantic highlighting for Go symbols, maps, JSON qualified types, routes, and response models
- Limited annotation highlighting to comment blocks that start after a `godoc` marker
- Added workspace-aware symbol resolution using VS Code symbol providers

## [0.0.1]

- Initial release with Swaggo annotation highlighting for Go comments
